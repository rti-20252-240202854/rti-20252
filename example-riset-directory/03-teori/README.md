# 03-teori

Folder ini berisi landasan teori, arsitektur penelitian, serta konsep-konsep yang digunakan dalam penelitian "Analisis Pengaruh Synthetic Minority Over-sampling Technique (SMOTE) terhadap Performa Algoritma Random Forest pada Deteksi Intrusi Menggunakan Dataset CIC-IDS2017".
Landasan teori digunakan sebagai dasar dalam memahami metode penelitian, menentukan alur eksperimen, serta mendukung analisis hasil penelitian.

## Isi yang diharapkan

Folder ini memuat dokumentasi mengenai:
Diagram alur penelitian (workflow penelitian).
Diagram proses klasifikasi menggunakan Random Forest.
Diagram skenario penelitian tanpa SMOTE (Baseline).
Diagram skenario penelitian menggunakan SMOTE (Treatment).
Konsep dasar Intrusion Detection System (IDS).
Penjelasan algoritma Random Forest.
Penjelasan metode Synthetic Minority Over-sampling Technique (SMOTE).
Penjelasan dataset CIC-IDS2017.
Penjelasan metrik evaluasi (Accuracy, Precision, Recall, F1-Score).
Penjelasan metode analisis statistik (Shapiro-Wilk Test, Wilcoxon Signed-Rank Test, Cohen's d, dan Confidence Interval).
