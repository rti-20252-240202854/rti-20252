# 02-literatur

Folder ini berisi referensi ilmiah yang digunakan sebagai dasar penyusunan penelitian "Analisis Pengaruh Synthetic Minority Over-sampling Technique (SMOTE) terhadap Performa Algoritma Random Forest pada Deteksi Intrusi Menggunakan Dataset CIC-IDS2017".
Studi literatur dilakukan untuk memahami perkembangan penelitian mengenai deteksi intrusi berbasis machine learning, mengidentifikasi research gap, serta menentukan metode yang digunakan dalam penelitian.

Tujuan Studi Literatur

Studi literatur dilakukan untuk:
Memahami perkembangan penelitian mengenai Intrusion Detection System (IDS) berbasis Machine Learning.
Mengidentifikasi kelebihan dan keterbatasan penelitian terdahulu yang menggunakan algoritma Random Forest dan teknik oversampling.
Menentukan metode yang sesuai untuk meningkatkan performa klasifikasi pada dataset yang memiliki ketidakseimbangan kelas (class imbalance).
Menemukan research gap sebagai dasar pelaksanaan penelitian.

## Topik referensi yang relevan

Referensi yang digunakan pada penelitian ini meliputi beberapa topik berikut.

Intrusion Detection System (IDS).
Machine Learning untuk deteksi intrusi jaringan.
Algoritma Random Forest.
Synthetic Minority Over-sampling Technique (SMOTE).
Dataset CIC-IDS2017.
Preprocessing data pada klasifikasi jaringan komputer.
Penanganan class imbalance pada machine learning.
Evaluasi model menggunakan Accuracy, Precision, Recall, F1-Score, Confusion Matrix, serta ROC-AUC.
Analisis statistik menggunakan Shapiro-Wilk Test, Wilcoxon Signed-Rank Test, Cohen's d, dan Confidence Interval.

Penelitian ini menggunakan beberapa jurnal utama sebagai acuan implementasi metode Random Forest, SMOTE, serta evaluasi performa pada dataset CIC-IDS2017.

| Penulis                                           | Tahun     | Keterangan                                                                                     |
| ------------------------------------------------- | --------- | ---------------------------------------------------------------------------------------------- |
| Sharafaldin, Lashkari, & Ghorbani                 | 2018      | Pengembang dataset CIC-IDS2017 yang digunakan sebagai dataset penelitian.                      |
| Chawla et al.                                     | 2002      | Penelitian asli yang memperkenalkan metode Synthetic Minority Over-sampling Technique (SMOTE). |
| Breiman                                           | 2001      | Penelitian yang memperkenalkan algoritma Random Forest.                                        |
| Penelitian terkait Random Forest + SMOTE pada IDS | 2021–2025 | Digunakan sebagai pembanding implementasi serta hasil penelitian.                              |

Selain jurnal utama tersebut, penelitian juga didukung oleh berbagai referensi lain mengenai Machine Learning, Intrusion Detection System, Random Forest, SMOTE, serta evaluasi statistik untuk memperkuat landasan teori dan pembahasan penelitian.
