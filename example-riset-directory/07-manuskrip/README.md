# 07-manuskrip

Folder ini berisi manuskrip ilmiah yang disusun berdasarkan hasil penelitian berjudul "Analisis Pengaruh Synthetic Minority Over-sampling Technique (SMOTE) terhadap Performa Algoritma Random Forest pada Deteksi Intrusi Menggunakan Dataset CIC-IDS2017".

Manuskrip merupakan ringkasan ilmiah dari laporan penelitian yang disusun mengikuti struktur artikel ilmiah, sehingga dapat disesuaikan dengan template jurnal nasional maupun internasional.

# tujuan manuskrip

Penyusunan manuskrip bertujuan untuk:

-Mendokumentasikan hasil penelitian dalam bentuk artikel ilmiah.

-Menyusun naskah yang siap dipublikasikan pada jurnal ilmiah.

-Menjadi dasar penyusunan artikel sesuai template jurnal nasional maupun internasional.

-Menyajikan hasil penelitian secara ringkas, sistematis, dan sesuai kaidah penulisan ilmiah.

-Menyampaikan hasil analisis mengenai pengaruh penerapan SMOTE terhadap performa algoritma Random Forest pada deteksi intrusi jaringan.

## Isi Manuskrip

Manuskrip memuat beberapa bagian utama, yaitu:

-Judul Penelitian

-Abstrak (Bahasa Indonesia dan Bahasa Inggris)

-Pendahuluan

-Tinjauan Pustaka

-Metodologi Penelitian

-Hasil dan Pembahasan

-Kesimpulan

-Daftar Pustaka

Seluruh isi manuskrip disusun berdasarkan hasil eksperimen menggunakan algoritma Random Forest, teknik Synthetic Minority Over-sampling Technique (SMOTE), serta dataset CIC-IDS2017.

## catatan
Catatan

Manuskrip ini merupakan ringkasan ilmiah dari laporan penelitian yang terdapat pada folder 08-laporan.

Seluruh metode, source code, dataset, hasil eksperimen, analisis statistik, tabel, gambar, serta pembahasan yang terdapat pada manuskrip berasal dari hasil penelitian yang telah didokumentasikan pada repository ini.

Apabila manuskrip akan dikirim ke jurnal tertentu, peneliti hanya perlu menyesuaikan format penulisan, gaya sitasi, serta template jurnal sesuai ketentuan penerbit tanpa mengubah substansi penelitian.
